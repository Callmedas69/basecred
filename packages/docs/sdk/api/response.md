# Response Schema

## UnifiedProfile

| Field        | Description                |
| ------------ | -------------------------- |
| identity     | Wallet address             |
| availability | Source states              |
| ethos        | Ethos facet (optional)     |
| talent       | Talent facet (optional)    |
| farcaster    | Farcaster facet (optional) |

## Availability States

- `available`
- `not_found`
- `unlinked`
- `error`
