# Development Phases

This document tracks **evolution**, not guarantees.

## Phase 1: Feasibility (Complete)

- Repository & type locking
- Ethos adapter
- Talent adapter
- Unified assembly
- Failure hardening
- Philosophical validation

## Phase 2: Score Expansion (Complete)

- Added `creatorScore`
- Added `creatorLevel` (`creator@v1`)
- `verifiedCreator` signal
- Multi-score `/scores` endpoint

## Phase 3: Ethos Refactor (Complete)

- Migrated to `/profiles` endpoint
- Timestamp support
- Backward compatible public API

## Phase 4: Time Semantics (Complete)

- `recency` classification
- `timeMeaning`
- `lastUpdatedDaysAgo`
- Immutable policies: `recency@v1`, `time@v1`

## Phase 5: Temporal Integrity (Deferred)

- Historical observations
- Stability classification

## Phase 6: Farcaster Account Quality (Draft)

Optional, opt-in Farcaster-only facet using Neynar User Score.
